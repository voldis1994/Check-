# CHECK — Nexus command center (1:1 desk)

Full sidebar: Dashboard, Accounts, Strategies, Trade Copier, Market Analysis,
Automation, Risk Manager, Reports, Alerts, Journal, Settings.

Hard numbers only (points / $ / count). Risk & automation = On/Off toggles.
No ATR. No percentages.

## Run

1. Put original MT4 in `Check\template\`
2. `Check\SETUP.bat`
3. Accounts → add → Risk Manager → LAUNCH MT4 → CHECK on M1
4. **Live Trading** ON (or PAPER)
